# 🎮 Análisis de Datos de Steam con SQL y DBeaver

Este proyecto es una exploración de datos de videojuegos disponibles en Steam, utilizando consultas SQL dentro del entorno de DBeaver. El objetivo es aplicar consultas básicas para comprender mejor los patrones de popularidad, reseñas y otros indicadores clave del mercado de videojuegos en esta plataforma.

---

## 📁 Dataset

El conjunto de datos utilizado fue obtenido desde Kaggle:

**Fuente**: [Steam Games Dataset on Kaggle](https://www.kaggle.com/datasets/whigmalwhim/steam-releases)

**Columnas destacadas**:

- `game`: Nombre del videojuego.
- `release`: Fecha de lanzamiento.
- `positive_reviews`, `negative_reviews`, `total_reviews`: Reacciones de los usuarios.
- `rating`: Calificación general del juego.
- `primary_genre`, `store_genres`: Géneros principales.
- `publisher`, `developer`: Empresa que publicó/desarrolló el juego.
- `players_right_now`, `24_hour_peak`, `all_time_peak`: Métricas de jugadores concurrentes.

---

## 🛠 Herramientas utilizadas

- **DBeaver**: Cliente SQL para consultar, explorar y transformar datos.
- **SQLite**: Motor de base de datos utilizado para almacenar el dataset.
- **GitHub Desktop**: Para control de versiones y subir el proyecto al repositorio.
